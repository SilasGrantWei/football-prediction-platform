# Support

Use public GitHub issues after the repository is published.

## Questions

Open a GitHub discussion or issue with:

- what you are trying to run
- operating system and shell
- Node.js and Python versions
- command output with secrets removed

## Bugs

Use the bug report issue template and include reproduction steps, expected
behavior, actual behavior, and relevant logs with secrets removed.

## Security

Do not report undisclosed vulnerabilities in public issues. Follow
`SECURITY.md`.
